# Claude Code Marketplaces

A collection of Claude Code plugin marketplaces.

## Available Marketplaces

| Marketplace | Description | Install |
|-------------|-------------|---------|
| [utilities](https://github.com/Yan-Yu-Lin/utilities) | Web search tools and utilities | `/plugin marketplace add Yan-Yu-Lin/utilities` |

## Cloning

To clone with all submodules:

```bash
git clone --recurse-submodules https://github.com/Yan-Yu-Lin/marketplaces.git
```

Or after cloning:

```bash
git submodule update --init --recursive
```
